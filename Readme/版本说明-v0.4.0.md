# 版本说明-v0.4.0

对应对象：基于 Gemini 的电商 AI 出图工作台

## 版本信息

- 版本号：`v0.4.0`
- 安装器：`IMAGE-STUDIO-WINDOWS-0.4.0.exe`

## 版本概述

`v0.4.0` 聚焦在“更完整的创作模式、更顺手的审核工作台、更统一的界面体验，以及更稳的协作链路”。

本说明面向对外交付与版本发布，描述当前版本可直接感知的能力，不展开内部修改记录。

## 本版本能力亮点

### 1. 创作模式补齐为完整工作流

- 支持标准出图、提示词模式、参考图复刻、套图模式、亚马逊 A+ 图五种创作模式
- 创建页的模式、比例、分辨率与参数胶囊重新整理，操作更集中
- 提示词链路更清晰，适合按市场、语言和平台快速发起批量任务

### 2. 审核与回看体验更完整

- 历史记录页支持更稳定的缩略图浏览，不再出现预览叠加
- 灯箱支持前后切换与独立查看入口，回看单张图片更顺畅
- 任务详情页更适合做结果筛选、审核和通过项导出

### 3. 模板、品牌与协作链路更适合团队使用

- 模板中心支持团队复用、扩展和按业务场景组织创作基线
- 品牌库可沉淀品牌语气、提示词约束和品牌偏好
- 飞书多维表格同步链路更稳定，适合做结果回写和团队协同

### 4. 设置与运行时控制更集中

- 设置页将 provider、sync 与运行控制收敛到一个统一控制台
- Gemini 官方接口、relay、中转测试和飞书连接测试更容易集中管理
- 素材目录、并发数与运行节奏配置更适合局域网和小团队部署

### 5. 对外交付更安全

- 继续采用版本化安装器命名，方便归档与回滚
- 发布链路支持清理默认密钥、任务记录和素材目录
- 更适合向协作者或客户环境做局域网部署交付

## 适用对象

- 电商运营团队
- 商品视觉与设计团队
- 需要局域网协作的内容生产团队
- 需要把生成结果沉淀到飞书表格的交付团队

## 使用提醒

- 安装后请先在设置页填写 Gemini 官方接口或兼容 relay 配置
- 如需启用飞书多维表格同步，请自行填写飞书接入参数并完成连通测试
- 安全发布版不包含本机 API Key、飞书密钥、本地任务记录和素材目录

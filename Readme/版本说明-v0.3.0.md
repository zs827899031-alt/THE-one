# 版本说明-v0.3.0

对应对象：基于 Gemini 的电商生图工作台

## 版本信息

- 版本号：`v0.3.0`
- 安装器：`IMAGE-STUDIO-WINDOWS-0.3.0.exe`

## 本次更新内容

### 1. 创作模式增强

- 保留并恢复“套图模式”，支持一键生成整套商品图
- 新增“亚马逊 A+ 图模式”，可生成完整 A+ 详情模块图
- 提示词模式新增“自动优化为真实照片”选项
- 参考图复刻模式提交流程修复，隐藏字段不再误拦截提交

### 2. 提示词与生成链路优化

- 输入文案会先翻译为目标语言后再参与图片生成
- 中文输出统一为简体中文
- 未输入的数据不再作为默认占位文本发送给模型
- 材质信息仅进入材质图相关提示词
- 尺寸与重量仅进入尺寸图相关提示词
- 尺寸图支持更适合目标市场的双单位表达

### 3. 飞书多维表格同步增强

- 飞书连接测试与字段映射体验优化
- 同一任务下多张成功图片会聚合到同一条飞书记录
- 支持大图与 4K 图片同步
- 支持失败任务重新同步飞书

### 4. 页面与交互优化

- 总览页压缩为更适合首屏查看的工作台布局
- 创作台参数区重排，图片类型、比例、分辨率更规整
- 提交任务入口移动到更高效的位置
- 历史记录页支持缩略图预览
- 历史记录页图片点击可直接放大查看
- 任务详情页改为更集中的工作台式浏览结构
- 结果区域新增“实际返回大小（MB）”显示

### 5. 兼容性与模型支持更新

- 按 Gemini 3.1 Flash Image Preview 最新官方文档同步宽高比与分辨率选项
- 支持 `0.5K / 1K / 2K / 4K` 分辨率档位
- 优化请求尺寸与实际返回尺寸的展示逻辑

### 6. 对外交付文档整理

- README 调整为更适合 GitHub 展示的对外交付版本
- 新增交付版使用说明
- 新增正式 PRD
- 删除不适合对外公开的安装器制作与内部开发说明

## 使用提醒

- 接收方安装后需要自行填写 Gemini 或中转配置
- 如启用飞书同步，也需要自行填写飞书接入参数
- 安装包默认不包含本机 API Key、飞书密钥、本地历史记录和素材目录

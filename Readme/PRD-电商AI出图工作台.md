# PRD-电商AI出图工作台

对应对象：基于 Gemini 的电商生图工作台

## 1. 项目背景

电商团队在日常运营中需要持续产出主图、场景图、卖点图、尺寸图、A+ 详情图等多类视觉素材。传统流程依赖人工设计与频繁返工，效率低、重复劳动多，且难以保证不同市场、语言、平台之间的输出一致性。

本项目目标是提供一个可在本地或局域网环境中运行的电商 AI 出图工作台，用于提升素材生产效率、降低重复操作成本，并增强结果管理与团队协作能力。

## 2. 产品目标

### 2.1 核心目标

- 让电商团队能快速生成多类型商品视觉素材
- 支持多国家、多语言、多平台场景下的内容适配
- 提供历史沉淀、结果审核与对外交付能力
- 支持飞书多维表格协作同步

### 2.2 交付目标

- 提供 Windows 可安装版本
- 支持非开发人员直接使用
- 发布包不携带本机敏感配置

## 3. 目标用户

- 电商运营
- 视觉设计
- 商品经理
- 跨境团队
- 使用飞书协作的内容团队

## 4. 典型使用场景

### 4.1 常规电商商品图生成

用户上传商品图后，生成多平台常用图片类型，如主图、场景图、卖点图、细节图等。

### 4.2 套图生产

用户按“套图模式”一次生成整套商品视觉，用于快速完成商品详情页面的主要素材准备。

### 4.3 亚马逊 A+ 图生产

用户按“亚马逊 A+ 图模式”生成完整的详情页模块图，覆盖主视觉、卖点解析、场景应用、尺寸标注等内容。

### 4.4 提示词直出

用户直接输入提示词进行概念图或真实摄影风格图片生成。

### 4.5 参考图复刻

用户上传参考图后，以参考图作为母版严格还原其构图、背景、光线、文字区、装饰和整体广告结构，只把图片原图里的主体及所有同类关联元素替换进去。

## 5. 功能范围

### 5.1 创作台

支持以下创作模式：

- 标准出图
- 套图模式
- 亚马逊 A+ 图模式
- 提示词模式
- 参考图复刻

支持以下参数能力：

- 主图上传
- 参考图上传
- 国家选择
- 输出语言选择
- 平台选择
- 比例选择
- 分辨率选择
- 数量选择
- 卖点、材质、尺寸、限制词、补充说明输入

其中参考图复刻模式是例外：

- 参考图只使用第一张
- 只保留比例、分辨率、数量三个生成参数
- 额外提供“文案来源”切换：`完全参照原图` / `按文案表格`
- 不再使用国家、语言、平台、模板、复刻强度、额外提示词、JSON 覆盖参与参考图复刻生成

### 5.2 生成与提示词处理

- 输入文案自动翻译为目标语言后再参与图片生成
- 材质信息仅进入材质图相关提示词
- 尺寸与重量仅进入尺寸图相关提示词
- 未输入的数据不发送给模型
- 中文输出统一为简体中文

### 5.3 历史记录

- 任务筛选
- 缩略图预览
- 图片点击放大查看
- 统计信息查看
- 进入详情页查看完整结果

### 5.4 任务详情

- 查看原图、参考图、生成图
- 查看请求比例、实际尺寸、实际返回大小
- 结果审核：入选、通过、淘汰
- 支持重新同步飞书

### 5.5 飞书多维表格同步

- 配置飞书接入参数
- 测试连接
- 字段映射
- 同任务单行、多图同格同步
- 支持大图上传

## 6. 核心页面

- 总览页
- 创作台
- 历史记录
- 模板中心
- 设置页
- 任务详情页

## 7. 业务规则

### 7.1 输出语言规则

- 所有自动翻译后的中文输出统一为简体中文
- 提示词模式下，用户直接输入的 prompt 不自动覆盖其原意，但会在启用真实照片优化时增加真实摄影导向

### 7.2 套图模式规则

固定生成以下六类图片：

- 主图
- 生活方式
- 卖点总览
- 场景图
- 材质工艺
- 尺寸参数

### 7.3 亚马逊 A+ 图模式规则

固定生成以下六类图片：

- 主视觉海报
- 核心卖点解析
- 多场景应用展示
- 产品细节
- 尺寸标注
- 文化价值

### 7.4 飞书同步规则

- 一个任务在飞书中对应一行
- 一个任务下的成功图片持续追加到同一图片字段
- 失败图片不上传，但统计信息会同步更新

### 7.5 参考图复刻规则

- 输入顺序固定：图1 是图片原图，图2 是参考图母版。
- 系统必须优先保持参考图不变，只替换主体与所有同类关联元素。
- “同类关联元素”包括包装、标签、产品家族变体，以及明显依附于主体的道具或图形元素。
- 默认文案来源为 `完全参照原图`：优先保留参考图原文案，仅在主体替换后出现明显冲突时做最小必要修改。
- 如果用户切换到 `按文案表格`：保持参考图文字版式、字体感觉和区块位置，改写内容优先使用创作台字段；字段缺失时回退到图片原图识别。
- 参考图复刻不再依赖市场语言参数驱动 prompt。
- 任务详情页不再展示参考图分析和海报槽位分析，仅保留最终结果、输入快照和实际发送 prompt。

## 8. 非功能要求

- 支持本地与局域网环境使用
- 支持 Windows 安装交付
- 默认发布包不携带敏感配置
- 对外使用者无需掌握开发环境
- 历史记录和详情页在常见桌面分辨率下保持可读

## 9. 验收标准

### 9.1 创作能力

- 用户能在创作台完成上传、参数选择和任务提交
- 套图模式和亚马逊 A+ 图模式能分别按固定模块生成
- 提示词模式支持真实照片优化选项

### 9.2 管理能力

- 用户能在历史记录页筛选任务并预览缩略图
- 用户能在详情页查看完整结果并进行审核
- 用户能导出通过项

### 9.3 协作能力

- 飞书连接测试通过后，可将成功生成的图片同步到多维表格
- 同任务多图可在同一条飞书记录中展示

### 9.4 交付能力

- 用户可通过 Windows 安装器完成安装与启动
- 安装后可进入设置页完成接口配置并正常使用

## 10. 当前交付边界

本次 PRD 对应的是当前可交付版本，重点覆盖：

- 本地运行与交付
- Gemini / relay 接入
- 创作台多模式生成
- 历史记录与详情页审核
- 飞书同步
- Windows 安装器交付

本文档不包含内部研发流程、打包脚本说明、安装器制作流程和后续版本迭代路线。

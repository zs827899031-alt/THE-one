# 版本说明-v0.3.1

对应对象：基于 Gemini 的电商 AI 出图工作台

## 版本信息

- 版本号：`v0.3.1`
- 安装器：`IMAGE-STUDIO-WINDOWS-0.3.1.exe`

## 本次更新内容

### 1. 参考图复刻稳定性修复

- 修复参考图复刻任务容易失败的问题，重建提示词改为更温和的单次发送逻辑
- 移除“严格提示词 + 回退提示词”双段展示，失败详情只保留实际发送给模型的一份提示词
- 修复切换国家时提示词模式、套图模式和参考图复刻模式被错误联动切换语言的问题

### 2. 分辨率与模型返回结果校验增强

- 修复请求 `4K` 但实际只返回低分辨率图片时仍被当作成功的问题
- 当返回图片明显低于目标档位时，系统会自动重试；多次不达标则按失败处理
- 结果详情页补充更清晰的尺寸档位、实际尺寸和返回大小展示

### 3. 创作台交互优化

- 提示词模式新增“自定义目标市场参数（可选）”，国家、语言、平台改为按需展开
- 提交任务入口上移，减少创作台底部的大块空白区域
- 图片类型、比例、分辨率按钮布局进一步规整，操作更集中

### 4. 历史记录体验优化

- 历史记录预览列支持点击放大查看缩略图，展示完整图片
- 统计区新增“生成耗时”，仅对已完成任务显示
- 历史记录中的预览仍保留多图缩略图展示，方便快速浏览生成结果

### 5. 提示词链路清理

- 中文输出统一约束为简体中文
- 未填写的空字段不再作为占位文本发送给模型
- 套图模式和 A+ 模式中，材质、尺寸等信息只进入对应图片类型的提示词

## 使用提醒

- 安装后请在设置页自行填写 Gemini 官方接口或兼容中转配置
- 如需启用飞书多维表格同步，请自行填写飞书接入参数
- 安全发布版不包含本机 API Key、飞书密钥、本地任务记录和素材目录
